# Alltrek Indonesia - Premium Outdoor Gear E-Commerce

A simple e-commerce website built with ReactJS, implementing 5 core pages:
1. **Homepage**
2. **Product Search Page (Catalog)**
3. **Product Detail Page**
4. **Cart Page**
5. **Checkout Page**

## Tech Stack
- **Core**: ReactJS (Vite)
- **Styling**: Vanilla CSS (CSS Variables)
- **Routing**: React Router DOM
- **State Management**: React Context API
- **Mocking**: Custom Asynchronous Mock API (Promise & setTimeout)

## How to Run the Project

### 1. Install Dependencies
Run the following command to install the required libraries:
```bash
npm install
```

### 2. Run in Development Mode
Run this command to start the local development server:
```bash
npm run dev
```
After starting the server, open your browser and navigate to the address shown in your terminal (usually `http://localhost:5173`).

### 3. Build for Production
To compile the application into static files optimized for production deployment:
```bash
npm run build
```
